01. Интерфейс -> 01. Главное меню

00. Введение

Слева от графической области IDE находится Главное меню.

Данное меню предназначено для взаимодействия с IDE. Изучим каждый раздел главного меню:

  1. Главное меню — включает в себя создание схем, открытие файлов, импорт схемы из игры «Берлога: Защита пасеки», сохранение текущего проекта в файл, а также готовые примеры для тренировки;
  2. Проводник — через это меню происходят действия с компонентами машины состояний, а также здесь находится иерархия состояний для удобной работы с элементами машины состояний;
  3. Компилятор — преобразование схемы машины состояний в формат выбранной целевой платформы;
  4. Загрузчик — загрузка скомпилированной схемы машины состояний в подключенное устройство;
  5. История изменений — возврат или отмена последних действий в редакторе IDE, а также их краткое описание;
  6. Настройки — содержит настраиваемые элементы IDE, например, тема приложения, выбор хоста и порта для компилятора и т.д.
