05. Раздел истории изменений

После того как пользователь создаст/откроет схему в разделе "истории изменений" отобразится две области:
Область "Undo Stack" содержит информацию о последних отменённых действиях пользователя, а область "Redo Stack" содержит информацию о произведенный пользователем изменениях с момента начала работы.

Кнопка "Назад" отменяет последнее действие пользователя с начало работы над схемой.

Кнопка "Вперёд" возвращает последнее отменённое действие пользователя.

Для быстрой отмены последнего действия, нажмите комбинацию клавиш Ctrl+Z.
Для быстрого возврата последнего отменённого действия, нажмите комбинацию клавиш Ctrl+Y.
