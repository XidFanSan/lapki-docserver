01. Интерфейс -> 01. Главное меню

01. Раздел меню

  • 1. Кнопка "Создать" - открывает модальное окно, где пользователю будет предложенно создать пустую схему, либо выбрать уже готовый шаблон в зависимости от платформы;

    Для  создания новой схемы, нажмите комбинацию клавиш Ctrl+N.

  • 2. Кнопка "Открыть" - открывается окно приложения, в котором пользователь должен выбрать файл схемы, которую планируется отредактировать;

    Для открытия схемы, нажмите комбинацию клавиш Ctrl+O.

  • 3. Кнопка "Сохранить" - сохраняет все  изменения схемы в тот же файл, если же файл был создан, то открывает окно приложения, где пользователь может выбрать место на устройстве куда сохранить схему, а также придумать название файлу;

    Для сохранения схемы, нажмите комбинацию клавиш Ctrl+S.

  • 4. Кнопка "Сохранить как..." - открывает окно приложения, где пользователь может выбрать место на устройстве куда сохранить схему, а также переименовать файл во время сохранения;

    Для сохранение схема как..., нажмите комбинацию клавиш Ctrl+Shift+A.

  • 5. Кнопка "Импорт..." - открывает окно приложения в котором пользователь может выбрать схему из игры «Берлога: Защита пасеки», чтобы отредактировать её в графической области IDE;

    Для импорта схемы из игры «Берлога: Защита пасеки», нажмите комбинацию клавиш Ctrl+I.

  • 6. Кнопка "Свойства" - открывает модальное окно, где пользователь может просмотреть свойства файла открытой схемы, а также изменить метаданные или добавить новые, такие как "name", "descrition" и т.д.
