03. Графическая область

В графической области до создания(открытия) схемы есть небольшое описание и список комбинации клавиш, которые облегчат начало работы в IDE.

После создания(открытия) схемы, в графической области отобразится редактор машины состояний, где пользователь непосредственно будет редактировать саму схему.
