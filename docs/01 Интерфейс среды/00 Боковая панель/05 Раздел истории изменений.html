05. Раздел истории изменений

После того как Вы создадите/откроете схему, в разделе "истории изменений" отобразится две области:
Область "Undo Stack" содержит информацию о последних отменённых Вами действиях, а область "Redo Stack" содержит информацию о произведенный Вами изменениях с момента начала работы.

Кнопка "Назад" отменяет последнее Ваше действие с начало работы над схемой.

Кнопка "Вперёд" возвращает последнее отменённое Вами действие.

Для быстрой отмены последнего действия, нажмите сочетание клавиш Ctrl+Z.
Для быстрого возврата последнего отменённого действия, нажмите сочетание клавиш Ctrl+Y.
