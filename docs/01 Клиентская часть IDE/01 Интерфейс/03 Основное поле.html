03. Основное поле

В основном поле до создания(открытия) схемы есть небольшое описание и список комбинации клавиш, которые облегчат начало работы в IDE.

После создания(открытия) схемы, в основном поле отобразится редактор машины состояний, где пользователь непосредственно будет редактировать саму схему.
