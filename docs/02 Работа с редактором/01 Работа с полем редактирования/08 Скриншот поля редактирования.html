08. Скриншот поля редактирования

Нажмите левой кнопкой мыши на поле редактирования, затем нажмите сочетание клавиш «Ctrl+P». Откроется окно приложения с выбором названия и пути сохранения скриншота. При выборе по заданному пути появится файл с заданным названием и расширением .png.
